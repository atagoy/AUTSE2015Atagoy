This plugin is to show user's recent search history.

== Description ==
If you have a search box on the site,
this plugin automatically records what logged In user's are searching and keep records.
simply displays the recent searches of logged In User and anonymous user.
Logged In user's history stored in the database and it's shows when that user is logged in.
Anonymous user's search history showing on the site until the browser closed.
This plugin added the widget to show user's recent search history.


== Installation ==
= From your WordPress dashboard =
*  Visit 'Plugins > Add New'
*  Search for 'User Recent Search History'
*  Activate 'User Recent Search History' from your Plugins page.
    

= Widgets: User's recent Search History =
Displays the recent search history of User.
