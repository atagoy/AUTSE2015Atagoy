Export a bzipped mysqldump output from WordPress admin interface.

== Description ==

A small plugin with a screen to dump the MySQL database to a bzip2 file and
download it. Useful for migrations and team development.

== Installation ==

1. Put the plugin in your `plugins` folder
2. Install it.
3. Click in "Settings > Export SQL File"
