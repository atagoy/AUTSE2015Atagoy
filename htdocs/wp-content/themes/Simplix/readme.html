Installation

  • 1. Unzip the template file (simplix.zip) that you have downloaded.
  • 2. Via FTP, upload the whole folder (Simplix) to your server and place it in the /wp-content/themes/ folder. Do not change directory name. The template files should thus be here now: /wp-content/themes/Simplix/index.php (for example).
  • 3. Check the CHMOD. For the files must be 644 and for the folders 755
  • 4. Log into your WP admin panel and click on the Design tab. Now click on the Simplix theme to activate it.
  • 5. Note: When first clicking the theme to activate it, the preview panel will show that the theme contains "errors", but this is only because you haven't set up the Theme Options yet.
  • 6. Complete all of the required inputs on the Simplix Options page (in the WP admin panel) and click "Save Changes". Section 2 below details this process.

Uploading Images for Thumnails

You could skip these points if you choose the First-Image on the Thumbnail assignment which means that images will be generated automatically. To have your thumbnail generated, simply add images in your post and the first-images will be used as thumbnails. If you choose Custom Post image Add a custom field in the post. The custom field key MUST be 'Image' (letter cases are important). Please reference the screenshot below and follow the subsequent instructions:

  • 1. On your Write > Post tab, complete all of the standard fields when adding a new post.
  • 2. Then, click on the "Add Media" icon and upload an image of your choice.
  • 3. Once uploaded, copy the complete "Link URL" field.
  • 4. Now scroll down to the Custom fields. Fill the custom field value with the path of your image. If you have the image located at http://www.yourblog.com/wp-content/uploads/2008/06/file_name.jpg then the key value of the custom field MUST be http://www.yourblog.com/wp-content/uploads/2008/06/file_name.jpg
  • 5. Click add custom field and publish post.
